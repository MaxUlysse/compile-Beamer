# compile-beamer [![Version][version-badge]][version-link] [![Works][works-badge]][version-link]

Just a simple beamer compiler to play with Nextflow and Docker

Usage:
   nextflow run main.nf --tex <file.tex> --theme <BTB || KI || SLL>
